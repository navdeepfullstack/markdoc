# 📚 Awaazghar - AudioBook Platform Documentation

Awaazghar is an innovative audiobook platform that brings stories, knowledge, and ideas to life through immersive audio experiences. Whether you're a reader on-the-go, visually impaired, or simply an audio content lover — Awaazghar offers a seamless and enriching way to consume books.

---

## 🚀 Features

- 🎧 **High-Quality AudioBooks**  
  Stream and download audiobooks in crystal-clear quality.

- 🔍 **Smart Search**  
  Find books by title, author, genre, or voice artist.

- 🎙️ **Multiple Narrators**  
  Experience books narrated by professional voice artists or AI-powered voices.

- 📥 **Offline Listening**  
  Download books to listen anytime, anywhere without internet.

- 📚 **Curated Collections**  
  Discover handpicked books in various languages and genres.

- 🧠 **AI-Personalized Recommendations**  
  Awaazghar learns your preferences to suggest books you'll love.

---

## 🛠️ Tech Stack

| Layer         | Technology Used         |
|---------------|--------------------------|
| **Frontend**  | Flutter / React Native (Mobile) |
| **Backend**   | Node.js / Express        |
| **Database**  | MongoDB / PostgreSQL     |
| **Storage**   | AWS S3 (Audio Files)     |
| **Streaming** | HLS / DASH               |
| **Authentication** | JWT / OAuth          |
| **Hosting**   | AWS / Vercel / Firebase  |
